FokusApp Rest Api

Options:

  • Accounts

      /accounts - GET Accounts data from database.
      /accounts - POST Accounts data into database with either XML or JSON..
      /accountDelete - Delete account entry using POST request with JSON or XML data
      /accountUpdate - Update account entry using POST request with JSON or XML data

  • Quotes

      /quotes - GET Quotes data from database
      /quotes - Post Quotes data into database with either XML or JSON.
      /quoteDelete - Delete quote entry using POSTrequest with JSON or XML data
      /quoteUpdate - Update quote entry using POST request with JSON or XML data

  • Notes

      /notes - GET Notes data from database.
      /notes - Post Notes data into database with either XML or JSON.
      /noteDelete - Delete note entry using POSTrequest with JSON or XML data

  • Events

      /events - GET Events data from database.
