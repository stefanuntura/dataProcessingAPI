FokusApp Rest Api

Options:

  • /accounts - GET or Posts Accounts data from database
  • /quotes - GET or Posts Quotes data from database
  • /notes - GET or Posts Notes data from database. Insert methodusing JSON is available via this link
  • /notesXml - GET or Posts Notes data from database. Insert methodusing XML is available via this link
  • /notes - GET or Posts Notes data from database. Insert methodusing JSON is available via this link
  • /notesDelete - Delete using GET or POSTrequest with JSON data
  • /notesDelete - Delete using GET or POSTrequest with XML data
  • /notesUpdate - Update note entry using POSTrequest with JSON data
  • /notesUpdateXml - Update note entry using POSTrequest with Xml data
  • /events - GET or Posts Events data from database. Delete method is available using Post.
