FokusApp Rest Api

Options:

  • /quoteInfo - Fetches all Quote data from database
  • /qutoteInfo?quoteID=writehere - Fetches only the Quote data from the database where quoteID matches quoteDay_Key

  • /appInfo - Fetches all App data from database

  • /notesInfo - Fetches all Notes data from database
  • /notesInfo?email=writehere& noteID=writehere - Fetches only the Notes data from the database where noteID matches note_id & email matches email

  • /scheduleInfo - Fetches all Schedule data from database

  • /scheduleUpdate?email=writehere&eventStatus=writehere - Updates the specific date&time status. write True for the eventStatus if the event has occured. False is default.

  • /uploadNotes - Upload a .txt type file
